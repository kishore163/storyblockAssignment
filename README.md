# Kishore’s Cricket Academy 

**Live Website:** https://storyblock-assignment.vercel.app

---

##  Overview
Kishore’s Cricket Academy is a React-based web application where **all content is managed through Storyblok** (Headless CMS). The project also supports **Storyblok Live Preview / Visual Editor**, enabling real-time preview of content changes while editing.

---

##  Tech Stack
- **Storyblok** — Headless CMS for content + assets
- **React.js** — Frontend UI
- **React Router** — Routing for a Single Page Application (SPA)
- **Redux** — State management


---

##  Key Features
- **High-quality images** served from **Storyblok Assets**
-  **Store** page with cricket products
- Ability to **add products to a wishlist**
-  **Help Center** section for FAQs
-  **Live editing / Visual Editor support** via Storyblok preview

---

## Run Locally

### 
1. Clone the repository
2. Install dependencies: npm install
3. Add Storyblok Access Token: create a .env file at root and add: VITE_STORYBLOK_TOKEN=your_storyblok_tokem
4. start the dev server: npm run dev
